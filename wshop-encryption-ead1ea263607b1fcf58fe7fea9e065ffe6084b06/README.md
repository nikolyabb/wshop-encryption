# Welcome to the Encryption Workshop!
This project is a very first intro for elementary school students to learn the problem statement and a bit of history of the data encryption.  

These materials will be used for:
* 15 minutes long presenation
* practical demo of encryption algorithm via this website(-s)
* small team game "pass the secret"
* foster enthusiasm to learn more for the students

### What you will find here
1. Web page to encrypt data
2. Another web page to decrypt data
3. Presentation slide deck

### Technical vision
1. Do GitHub action builds for the apps as a static code
2. Auto-deploy the code to the Internet
3. Help contributors to learn collaboration on GitHub on ther real projects

### How to contribute
1. Do a fork of this repo to your profile
2. Find a ticket on Issues page and come up with solution
3. Open Pull Request from your repo to `develop` branch of this one

Good luck, and happy learning!
